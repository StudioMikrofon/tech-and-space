Anthropic's routine release of Claude Code's 2.1.88 update turned into a major leak, exposing over 512,000 lines of code and 2,000 TypeScript files. The leak, which was uploaded to a public GitHub repository, has been copied more than 50,000 times, giving the entire internet and Anthropic's competitors a chance to examine the code. According to [Ars Technica](https://arstechnica.com), the leak revealed some of the features Anthropic has been working on, including a 'Proactive' mode for Claude Code.


The leak has sparked a mixture of reactions, with some [developers](https://github.com) praising the code's quality and others raising concerns about the potential implications of such a large leak. As [The Verge](https://www.theverge.com) notes, the leak has given Anthropic's competitors a unique opportunity to examine the company's code and potentially gain a competitive advantage.

The community is responding with a mix of shock and curiosity, with some users [reporting](https://github.com) that they have found evidence of upcoming features in the leaked code.


For all the noise, the actual story is that Anthropic's competitors now have a chance to catch up, and the company's lead may be shrinking.
